Embed Github Events Widget «alpha version»

Configuration:

Github username:without @, user or organisation
Events count:1-30, default=5
Skin: matching your website
Display authors: Should be autors of events displayed? default=always
Allowed event types:comma separated list of GH event types
Custom CSS style:add your own CSS rules, use dev-tools to discover structure. For example: 
.event{background-color:orange}

Use this code:

Do you want to host your scripts by yourself? Download the library, place it somewhere in your webserver and adapt src value of the script above.
